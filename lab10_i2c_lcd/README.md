# Lab 10 — I2C LCD Interface

Проект работы с интерфейсом I2C: сканирование шины, проверка устройства и вывод текста на LCD-дисплей.

## Что делает проект

Проект позволяет:
- сканировать I2C-устройства;
- проверять наличие отклика на шине;
- выводить текст на LCD;
- очищать экран;
- переключать режимы отображения;
- управлять проектом через UART.

## Используемая периферия

- I2C1
- USART2
- GPIO

## Основные команды

| Команда | Назначение |
|---------|-------------|
| HELP | Показать список команд |
| SCAN | Сканировать I2C-шину |
| TESTLCD | Проверить подключение дисплея |
| TEXT ... | Вывести текст на экран |
| CLEAR | Очистить дисплей |
| STATUS | Показать состояние |

## Назначение проекта

Работа демонстрирует настройку и использование I2C-интерфейса для обмена данными с внешним дисплеем.
