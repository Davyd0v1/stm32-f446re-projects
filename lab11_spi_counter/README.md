# Lab 11 — SPI Counter

Проект управления счётчиком через SPI и сдвиговый регистр.

## Что делает проект

Проект позволяет:
- передавать данные через SPI;
- отображать счётчик на выходе;
- запускать и останавливать счёт;
- менять скорость обновления;
- задавать значение вручную;
- управлять работой через UART-команды.

## Используемая периферия

- SPI1
- USART2
- GPIO

## Основные команды

| Команда | Назначение |
|---------|-------------|
| HELP | Показать список команд |
| STATUS | Показать текущее состояние |
| SET x | Установить значение |
| INC | Увеличить значение |
| DEC | Уменьшить значение |
| COUNT START | Запустить счётчик |
| COUNT STOP | Остановить счётчик |
| SPEED x | Изменить скорость обновления |

## Назначение проекта

Работа демонстрирует обмен по SPI и управление внешним устройством через передачу байтов данных.
