# Lab 12 — WWDG Watchdog Demo

Проект демонстрации работы оконного сторожевого таймера WWDG на STM32F446RE.

## Что делает проект

Проект позволяет:
- запускать и останавливать WWDG;
- изменять окно и значение счётчика;
- обновлять watchdog вручную;
- выводить причину сброса;
- имитировать зависание системы;
- управлять режимами через UART.

## Используемая периферия

- WWDG
- USART2
- GPIO
- EXTI

## Основные команды

| Команда | Назначение |
|---------|-------------|
| HELP | Показать список команд |
| STATUS | Показать состояние системы |
| WWDG ON | Включить watchdog |
| WWDG OFF | Выключить watchdog |
| WINDOW x | Установить значение окна |
| COUNTER x | Установить значение счётчика |
| KICK | Обновить watchdog |
| CAUSE | Показать причину сброса |
| HANG | Имитировать зависание |
| BUZZ ON | Включить buzzer |
| BUZZ OFF | Выключить buzzer |

## Назначение проекта

Работа демонстрирует принципы защиты программы от зависания с помощью оконного сторожевого таймера.
