# Lab 8 — PWM Controller

Проект управления PWM на STM32F446RE через UART-команды.

## Что делает проект

Проект позволяет:
- запускать и останавливать PWM;
- изменять частоту;
- изменять скважность;
- выполнять плавное изменение параметров;
- управлять режимами работы через UART;
- выводить текущий статус в терминал.

## Используемая периферия

- TIM1
- TIM2
- TIM3
- USART2
- GPIO

## Основные команды

| Команда | Назначение |
|---------|-------------|
| HELP | Показать список команд |
| STATUS | Показать текущее состояние |
| START | Запустить PWM |
| STOP | Остановить PWM |
| FREQ x | Установить частоту |
| DUTY x | Установить скважность |
| FADE a b s | Плавное изменение параметра |

## Назначение проекта

Работа демонстрирует настройку таймеров в режиме PWM и взаимодействие с пользователем через UART-терминал.
