# Lab 9 — ADC + UART Control

Проект чтения аналогового сигнала через ADC с выводом данных в UART и управлением режимами работы.

## Что делает проект

Проект позволяет:
- считывать значение ADC;
- выводить текущее значение в UART-терминал;
- запускать и останавливать измерения;
- задавать пороговые значения;
- изменять чувствительность;
- управлять светодиодами и buzzer;
- отображать состояние системы.

## Используемая периферия

- ADC1
- USART2
- GPIO

## Основные команды

| Команда | Назначение |
|---------|-------------|
| HELP | Показать список команд |
| START | Запустить измерения |
| STOP | Остановить измерения |
| VALUE | Показать текущее значение ADC |
| STATUS | Показать состояние системы |
| THRESHOLD x | Установить порог |
| SENS x | Установить чувствительность |
| RESET | Сбросить настройки |

## Назначение проекта

Работа демонстрирует использование АЦП, обработку измеренных значений и управление устройствами через UART-интерфейс.
